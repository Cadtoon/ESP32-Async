The Access Point settings are similar to the network settings. Although you won't have to worry about reserving an IP, the SIDD name should be a unique name.

The default IP, 192.168.4.1, can be changed but has not been added to this App.

Minutes to Live is the amount of time the AP will be available to connect to when first booted. This time is only initiated after boot up. If the timer is set to 3 minutes and has 2:31 remaining, and you change it to 5 minutes, there will now be 4:31 remaining. It will remain active until the time expires.

Again the "Restart ESP" button is there as a convenience. Any changes made to the name or password won't be initiated until the ESP is restarted or the AP is reinitialized. You will need to reconnect to the AP.

NOTE: The minimum time it can be changed to in this webpage is 3 minutes. If you edit this webpage and change the min range value, a setting of 1 minute may not give you enough time to make changes before it times out.
