The Access Point settings are similar to the network settings. Although you won't have to worry about reserving an IP, the SIDD name should be a unique name.

The default IP 192.168.4.1 can be changed but has not been added to this App.

Minutes to Live is the amount of time the AP will be available to connect to. This time is only initiated during start up or after the network connection has been lost. If the timer is running, a change is immediate. If the timer is set to 3 minutes and has 2:31 remaining, and you change it to 5 minutes, there will be 4:31 remaining. It will remain active until the time expires and a network connection is established. Should the network connection continue to fail, the AP will continue to run, and shutdown is diverted. When the time expires, it will disconnect immediately if the network is established or reestablished.

Again the "Restart ESP" button is there as a convenience. Any changes made to the name or password won't be initiated until the ESP is restarted or the AP is reinitialized. You will need to reconnect to the AP.
