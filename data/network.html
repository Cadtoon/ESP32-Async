NOTE: You MUST click the Update button for each row to apply a change. If you fill in more than one field, then click on an Update, only the field in the same row as the Update button you clicked will be updated.

These settings are required to connect to your Wi-Fi Router. After you have updated the fields your change is displayed in the grey area. After verifying each grey area is correct, you will need to restart the ESP.

The timer is used to check the Wi-Fi status during normal operation, by default every 2 minutes. After establishing a connection, you may wish to change this value up to 60, checking the network connection once every 60 minutes. You can set it anywhere between 1 and 60. I recommend 10-15.

Once a local Wi-Fi connection is successful, the network IP will be displayed just above the Menu on each webpage. When not connected it displays 0.0.0.0.

The Access Point (AP) is started on a reboot or restart. If there is no local Wi-Fi connection, the timer isdisabled and the AP is alway activated. If there is a local Wi-Fi connection, it will run for the minutes set in the AP setting "Minutes to Live” and stays running until the time has elapsed. If you are connected to the AP, 192.168.4.1, and the AP disconnects before changes have been completed, you will need to restart the ESP to access the AP again, or simply use the local IP address.

The Hostname you provide will help identify this ESP in the router. Do not use spaces in the name. The title on the top of all the webpages is also this Hostname. This name SHOULD be unique to your network. If you use a Hostname more than once you will not be able to reserve an IP for it in your router settings.

Be sure your new setting is displayed correctly in the grey area after updating a change. After all changes are completed restart the ESP. When you return the local Wi-Fi IP is listed above, disconnect your device from the Wi-Fi AP and reconnect to your local Wi-Fi. Open your browser to the given IP using http://.
