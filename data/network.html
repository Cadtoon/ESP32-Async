These settings are required to connect to your Wi-Fi Router. You can change any of these settings without having to restart the ESP. When the ESP is powered up, the AP is started for the minutes set in the Access Point setting "Minutes to Live” and stays running until the time has elapsed. Before it is shutdown it confirms there is a network connection. If not, the shutdown is diverted until the network connection is confirmed. The AP function runs every Loop cycle. When the TTL has expired and the AP has been shut down, the next statement checks the network connection. If it fails, it attempts to restart the network and restarts the AP.

The App is setup to use the DHCP supplied by your router. If you require a static IP, you will need to add some code to provide that option. Typically, most routers can be setup to assign, or reserve, IPs by MAC address.

The Hostname you provide will help identify this ESP in the router. The title on the top of all the webpages is also this same Hostname. This name SHOULD be unique to your network. If you use a Hostname more than once you will not be able to reserve an IP for it in your router settings.

The Restart button is provided as a convenience. One scenario when you will need to use it, is if you have two Wi-Fi routers within the ESPs Wi-Fi range, and you want to change routers the ESP is connected to.

There is a little hidden spot in the ESP that remembers the last known connected network. So if you make a mistake changing your settings, the ESP will remember what worked before and will attempt to connect using those settings again.
