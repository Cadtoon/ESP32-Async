These settings are required to connect to your Wi-Fi Router. You can change any of these settings without having to restart the ESP. When the ESP is powered up, the AP is started for the minutes set in the Access Point setting "Minutes to Live” and stays running until the time has elapsed. Before it is shutdown.

The App is setup to use the DHCP supplied by your router. If you require a static IP, you will need to add some code to provide that option. Typically, most routers can be setup to assign, or reserve, IPs by MAC address.

The Hostname you provide will help identify this ESP in the router. The title on the top of all the webpages is also this Hostname. This name SHOULD be unique to your network. If you use a Hostname more than once you will not be able to reserve an IP for it in your router settings.

A restart is required to apply changes made in the Network Station settings. Be sure your new settings are displayed correctly in the grey area. The Restart button is provided as a convenience.
