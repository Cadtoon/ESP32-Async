Congratulations, you made it this far! Welcome to your "%inputNetHOST%" Web-configuration Portal!

Use the orange Menu above to navigate through the various settings provided by the "%inputNetHOST%" App now installed on your ESP.

Other than displaying the actual ESPs time on the pages the sketch itself doesn't have any sensor functionality. It has been put together to help users wanting to use ESPAsyncWebServer, AsyncTCP, and AsyncElegantOTA, in an all-in-one "example" sketch developed using the Arduino IDE.

The next steps is to configure the Access Point and Station settings. The NTP server and time zone are set to my location which is Mountain time. Indeed, you can set the time zone for anywhere in the world and links are provided to help you look them up.

Additional information is provided on each webpage. Let me know about your project and how you will be using this App. You can send me an email to r.j.pearn@gmail.com. I might to have to setup my own "Buy me a Coffee" webpage and get some free beers!

The table below is used on the webpages with settings.

  • The yellow area is a description of the setting.

  • The area in grey displays the existing value for that setting. In setting with sliders or switches, when a change is made the value in this area will also change.

  • The area in white will contain an input field, slider or switch.

  • The green area contains an Update button. No changes are saved until this button is clicked. Once updated the value in the grey area will be updated.

  • The red area contains a "Restart ESP" button. The only indication it has been clicked is the clock at the top of the page will stop. The clock will start again once the ESP has restarted and is running, providing that the ESPs IP address hasn't changed.


       
       

If you have the sketch open in Arduino IDE and the ESP connected to the USB, open the Serial Monitor. Restart the ESP. After initialization the APP lists all the settings loaded from the files. Good way of finding out what a forgotten password is. It will then display the IP address for both the Station and Access Point. The Web Server is started and a "Webserver Ready!" message is shown.

The next message displayed is "AP will disconnect in xx:xx" xx:xx depends on the minutes to live setting. This message is displayed once every second until the AP disconnects.

When a webpage is opened, a date and time message will appear. The date every minute and the time every second. Open two webpages at the same time will display two dates and two times, three webpages, three of each message. Every time a page calls for an update it is displayed in the serial monitor. Change a setting, the change will be displayed and all the values in the saved files are again loaded and listed.

One last note of advice. As you will see there really isn't that many settings and I could have listed them all on one page. The problem is the ESP cannot server the page fast enough and the page is noticeably slow regenerating.
