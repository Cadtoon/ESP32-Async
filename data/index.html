Congratulations, you made it this far! Welcome to your "%inputNetHOST%" Web-configuration Portal!

Use the orange Menu above to navigate through the various settings provided by this App now installed on your ESP.

Other than displaying the actual ESPs time on the pages the sketch itself doesn't have any sensor functionality. It has been put together to help users wanting to use ESPAsyncWebServer, AsyncTCP, and AsyncElegantOTA, in an all-in-one "example" sketch developed using the Arduino IDE.

Your next step is to configure the Network Station and Access Point settings. The NTP server and time zone are set to my location which is Mountain time. Indeed, you can set the time zone for anywhere in the world and links are provided to help you look them up.

Additional information is provided on each webpage. Let me know about your project and how you will be using this App. You can send me an email to r.j.pearn@gmail.com.

The table below is used on the webpages with settings.

  • The yellow area is a description of the setting.

  • The area in grey displays the existing value for that setting. In setting with sliders or switches, when a change is made the value in this grey area will also change. With some browsers it might be difficult to read.

  • The area in white will contain an input field, slider or switch.

  • The green area contains an Update button. No changes are saved until this button is clicked. Once updated the value in the grey area will be updated. NOTE: When you enter data into a field, click on the Update button in the same row before entering data into another field. If you enter data into 2 fields and click on an Update button, only the value entered into the same row as the button will be kept, the other field data will be lost.

  • The red area contains a "Restart ESP" button.


       
       

If you have the sketch open in Arduino IDE and the ESP connected to the USB, open the Serial Monitor. Restart the ESP. There are many messages provided.
