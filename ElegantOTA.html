This is the AsyncElegantOTA App. I have no control over its functionallity. It only sucessfully uploads bin files. If you upload any other type of file it will corrupt the SPIFFS file system and all the existing saved vars will be deleted. You might even have to reformat the SPIFFS to get it back.

This tutorial will show you how to create a bin file of a sketch using Arduino iDE. https://randomnerdtutorials.com/esp32-over-the-air-ota-programming/. Be sure you have selected "Firmware" when uploading your sketch.

This tutorial will show you how to create a bin file of the files located in a sketches data folder using Arduino iDE. https://randomnerdtutorials.com/esp32-web-server-spiffs-spi-flash-file-system/ Be sure you have selected "Filesystem" when uploading your data.

Here the OTA has been placed into an iframe. For quicker access you can go directly to http://yourIPaddress/update, this is its default location.
